## ✨ Using the chat features

The chat interface includes tools to help you manage your conversations.

### Streaming messages
Messages appear as they generate. You see the text grow in real-time. This reduces wait times and keeps the user interface responsive.

### Slash commands
Type a forward slash followed by a keyword to trigger specific actions. For example, use /clear to wipe the chat history or /export to save your current transcript to a text file.

### Artifact panels
The software displays complex information in fixed windows. This keeps your chat clean while allowing you to view detailed tables, code snippets, or long documents side-by-side with your conversation.

### Command palette
Press the Command and K keys at the same time to see the search menu. Use this menu to jump between active chats or run system commands without using your mouse.

## 📋 Frequently asked questions

### Does the application track my data?
The tool acts as a bridge between your computer and your database. It does not store private data on our servers. You maintain control of all information that passes through the interface.

### Can I change how the chat looks?
Yes. You can edit the themes in the settings menu. You can switch between light and dark modes.

### What do I do if the software freezes?
Close the program entirely using the Task Manager if the window stops responding. Restart the process to clear the temporary memory. If problems persist, reinstall the application using the latest version from GitHub.

### Does this work offline?
The interface requires a connection to a backend service to generate responses. It cannot function without an active network link.
